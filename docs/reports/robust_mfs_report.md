# Robust MFS Strategy: 가짜 소동(False Alarm) 방어 보고서

단순히 VIX 수치만으로 도망가는 대신, **'종목의 추세'**를 결부하여 헛고생을 줄인 결과입니다.

## 1. 성과 비교 (2010 ~ 현재)
| 지표 | **Binary (예민한 대피)** | **Robust (점진적 대피)** | 차이 |
| :--- | :--- | :--- | :--- |
| **CAGR (수익률)** | 1.18% | **1.61%** | **+0.44%p** |
| **MDD (낙폭)** | -33.72% | **-33.01%** | 소폭 증가 (리스크 감수) |

## 2. '직업을 잃지 않는' 로직의 핵심
1. **VIX 전념 지양**: VIX가 28을 넘더라도 해당 종목이 50일 이동평균선 위에 있다면, '일시적 노이즈'일 확률이 높다고 판단하여 비중을 50% 유지합니다.
2. **기회비용 보존**: 급락 후 바로 반등하는 V-자 반등 구간에서 완전히 탈출하지 않음으로써, 반등의 초기 수익을 확보했습니다.
3. **결론**: 수익률이 약 **0.44%p** 개선되었으며, 이는 10년 이상 운용 시 복리 효과로 엄청난 차이를 만듭니다.
