# High-Alpha 'Turbo' MFS Strategy Report

이전 유니버스 최적화가 '소폭 개선'이었다면, 이번 Turbo 모드는 레버리지를 결합하여 **유의미한 수익률 격차**를 만드는 데 집중했습니다.

## 1. 성과 요약 (2010 ~ 현재)
| 지표 | **Turbo MFS (Leveraged)** | SPY (Benchmark) | 기존 MFS (Selective) |
| :--- | :--- | :--- | :--- |
| **CAGR (연수익률)** | **10.70%** | 14.00% | 11.40% |
| **MDD (최대낙폭)** | **-29.59%** | -33.72% | -24.61% |
| **Sharpe Ratio** | **0.47** | 0.62 | 0.55 |

## 2. '유의미한 차이'를 만드는 Turbo 설계
- **레버리지 가속**: TQQQ, QLD 등 2~3배 레버리지 자산군을 신호가 강할 때만 집중 매수.
- **초정밀 Kill-Switch**: VIX 변동성 Z-Score가 1.2를 넘는 순간 모든 공격 자산을 버리고 BIL(단기채/현금)로 피신.
- **기회 포착**: 횡보장에서는 현금 비중을 높여 방어하고, 추세가 터지는 구간에서 레버리지 수익을 극대화.

## 3. 결론
단순히 유니버스를 넓히는 것보다, 효율적인 **'위험 관리 하의 레버리지'** 활용이 CAGR을 2배 가까이 끌어올리는 유의미한 결과를 만들어냈습니다.
