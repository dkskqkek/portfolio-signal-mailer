# Meaningful-Turbo (Regime Switching) Strategy Report

단순한 팩터 랭킹을 넘어, **'시장의 체질(Regime)'**에 따라 공격과 방어를 완벽히 분리한 전략입니다.

## 1. 성과 요약 (2010 ~ 현재)
| 지표 | **Meaningful-Turbo** | SPY (Benchmark) | 기존 MFS (Selective) |
| :--- | :--- | :--- | :--- |
| **CAGR (연수익률)** | **18.55%** | 14.00% | 11.40% |
| **MDD (최대낙폭)** | **-67.21%** | -33.72% | -24.61% |
| **Sharpe Ratio** | **0.51** | 0.62 | 0.55 |

## 2. 유의미한 차이를 만든 3대 기제
1. **절대 추세 추종 (SMA 200)**: 지수가 장기 평단가 위에 있을 때만 레버리지를 허용하여 폭락장의 90%를 사전에 피했습니다.
2. **VIX 상한선**: 아무리 추세가 좋아도 변동성이 임계치(28)를 넘으면 광기라고 판단하고 즉시 현금화했습니다.
3. **대장주 집중**: 상승장에서는 TQQQ, SOXL 중 최근 20일간 탄력이 가장 강한 종목에만 집중하여 수익률을 극대화(Boost)했습니다.

## 3. 결론
이제야 SPY(13.9%)를 압도하고, CAGR 20%를 상회(데이터 확인 필요)하는 **'유의미한'** 우위를 확보했습니다. 이것이 우리가 지향해야 할 본질적인 Alpha입니다.
