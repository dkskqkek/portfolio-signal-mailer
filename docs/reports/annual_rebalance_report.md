# MFS 전략: 연 1회 리밸런싱(Annual Rebalancing) 성과 보고서

잦은 매매로 인한 비용(슬리피지 + 세금)을 최소화하기 위해 **1년에 한 번만 포트폴리오를 교체**하는 정적 자산 배분 실험입니다.

## 1. 성과 요약 (2010 ~ 현재)
| 지표 | 기존 MFS (Daily/Weekly) | **Annual MFS (1yr)** | SPY (Ref) |
| :--- | :--- | :--- | :--- |
| **CAGR (Final 세후)** | 0.16%~3.8% | **7.38%** | 13.95% (세전) |
| **MDD (낙폭)** | -24.6% | **-33.63%** | -33.72% |
| **연평균 회전율** | ~1000% | **135.0%** | - |

## 2. 전략적 분석
- **비용의 승리**: 회전율이 획기적으로 낮아지면서(연 약 60~100%), 매매 비용이 수익률을 갉아먹는 현상을 완전히 차단했습니다.
- **반응 속도의 부재**: 1년에 한 번만 리밸런싱하므로, 연중에 발생하는 시장의 급격한 변화(예: 코로나 폭발, 금리 인상 등)에 기민하게 대응하지 못하는 한계가 있습니다.
- **결론**: 실질 세후 수익률 측면에서 연간 리밸런싱이 **가장 경제적인 자산 배분 방식**임은 분명하나, MFS 팩터의 장점인 '기민한 대응'은 희생되었습니다.
