# MAMA Lite: RL 비중 최적화 성과 보고서 (Refined)

사용자 피드백과 시스템 분석을 반영하여 시계열 분리 학습 및 거래 비용을 포함한 결과입니다.

## 1. 학습 및 테스트 요약
- **총 학습 스텝**: 30,000 steps
- **학습 기간**: 2019-01-02 ~ 2022-12-30
- **테스트 기간 (Out-of-Sample)**: 2023-01-03 ~ 2026-01-23
- **테스트 최종 NAV**: 2.1530

## 2. 주요 개선 사항 반영
1. **GNN 실전 연동**: QQQ 단일 지수가 아닌 GNN이 선정한 Top 3 종목의 실시간 수익률을 Alpha 자산으로 반영.
2. **거래 비용 현실화**: 비중 변경 시 0.1%의 거래 비용(Transaction Cost)을 보상 함수에 페널티로 부과.
3. **과적합 방지**: 2023년 이후 데이터를 학습에서 제외하여 순수한 Out-of-Sample 성능 검증.
